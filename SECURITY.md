# Security

Report suspected vulnerabilities to security@growdirect.io. Do not
open public issues for security matters.

This vault is documentation-only; it contains no executable code and
no secrets. Confidentiality is enforced by access control (NDA or
acknowledgment), not by code-level security controls.

Canary Retail product security — automated vulnerability scanning,
SAST, git-history secret scanning, OWASP Top 10 code-level review,
pending formal penetration test — is documented in the Canary
product repository.

Secret and credential management policy — tiered dev / CI /
production storage, rotation, AWS Secrets Manager roadmap — lives
in the GrowDirect platform repo at
`docs/security/key-management.md`.
