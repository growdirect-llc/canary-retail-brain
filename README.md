# Canary Retail — Brand & Architecture Brain

Canary Retail is GrowDirect LLC's retail operating system for small
and mid-sized specialty retailers with online footprint. POS-agnostic
by architecture. ARTS-standards native. Multi-tenant SaaS.

This vault holds the product brand, the architecture, and the
externalized case studies. It is externally-shareable under NDA or
access acknowledgment.

## What lives here

- **Platform** — positioning, the 13-prefix spine, the canonical
  retail data model (CRDM), ARTS adoption, the Differentiated-Five
  add-on layer.
- **Modules** — one article per module in the spine (C, D, F, J, S,
  P, T, R, N, L, Q, W, A).
- **Integrations** — POS adapters, payments, ecommerce, security
  hardware, MDM/ITAM.
- **Architecture** — service mesh, transaction pipeline, detection
  engine, case management, evidence chain.
- **Case studies** — abstracted patterns by vertical (SMB specialty
  archetype, multi-store apparel, F&B, sporting goods). No named
  clients.
- **Roadmap** — versioned milestones from v1 (Differentiated-Five)
  through v3 (full spine).

## What does NOT live here

- Runtime code (that's in `~/GrowDirect/Canary/`)
